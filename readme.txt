Plugin adds aditional buttons to media upload module allowing of faster images edit and add to post.

== Description ==

Plugin adds aditional buttons to media upload module:
  
1. Add alt text to all uploaded photos,
 
2. Insert all photos into post at cursor position with clear markup.

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page 

or 

use "Wordpress Add New Plugin" option and search for: "Image Photoroll Creator For Photographers", then install and activate the Plugin
